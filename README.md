# Landecker Archive

A collaborative repository for the historical and etymological study of the Landecker surname, with a focus on its origins in Landeck, Western Prussia, and its unique symbolic associations.

## Project Purpose

This repository serves as a living document for historians, genealogists, and researchers interested in the Landecker surname. The project aims to:

- Document the etymology and historical development of the surname
- Track geographic origins, particularly Landeck in Western Prussia
- Explore symbolic associations, such as the unicorn in heraldic references
- Provide a platform for collaborative research and source sharing

## Structure

- **Research files**: Detailed analyses of specific topics (e.g., unicorn symbolism, migration patterns)
- **Source citations**: References to historical documents, parish records, and heraldic compendia
- **Contributor guidelines**: Instructions for adding new research
